# 야구매니저 — Claude Code 가이드 저장소

## 지침

- 모든 응답은 **한국어**로 한다.

## 프로젝트 개요

BCM(AroTechBCM)에서 제작 중인 **야구매니저** 프로젝트의 Claude Code 도입 가이드 모음입니다.
직군별 경력 개발자가 AI 코딩 도구를 처음 사용할 때 참고할 수 있도록 작성되었습니다.

| 직군 | 가이드 파일 |

| Unity 클라이언트 개발자 (C#) | `unity-claude-code-guide.md` |
| Unreal Engine 개발자 (C++) | `unreal-claude-code-guide.md` |
| Unreal 그래픽스 개발자 | `unreal-graphics-claude-code-guide.md` |
