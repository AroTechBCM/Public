# VS Code + Claude Code 매뉴얼 요약

> 원본: `vscode.md`
> 대상: BCM 개발팀 / 야구매니저 프로젝트
> 버전: 1.1 (2026.03)

---

## 핵심 개념

일반 AI 채팅과 달리 VS Code + Claude Code는 **프로젝트 폴더 전체를 맥락으로 활용**하여 파일을 직접 생성·수정한다. 프로그래밍 지식 불필요.

---

## 설치 (2단계)

1. [VS Code](https://code.visualstudio.com) 설치 — PATH 추가 옵션 체크
2. VS Code 확장 검색 → **Claude Code (게시자: Anthropic)** 설치 → 첫 실행 시 Anthropic 계정 로그인

---

## 기본 사용법

- 왼쪽 사이드바 **✱ Spark 아이콘** 클릭 → 채팅 패널 열기
- 한국어로 자연스럽게 요청 입력
- 파일 수정 시 **diff 화면에서 Accept / Reject** 으로 확인 후 적용

## CLAUDE.md — AI 규칙 파일

프로젝트 루트에 위치하며, Claude가 대화 시작 시 가장 먼저 읽는 파일.

포함 권장 항목:
- 응답 언어 (한국어)
- 문서 형식 (마크다운)
- 게임 핵심 구조 (등급, 포지션, 스탯 등)
- 기획서 작성 양식
- 수치 설계 원칙

> 수정 후 반드시 **새 대화 시작** 해야 반영됨

---

## 자주 쓰는 프롬프트 템플릿

| 작업 | 템플릿 |
|---|---|
| 새 기획서 | `[폴더]에 [주제] 기획서를 작성해줘. [참고 문서]를 참고해서 일관성 있게.` |
| 문서 수정 | `[파일명]에서 [섹션]을 [수정 내용]으로 변경해줘.` |
| 수치 공식 | `[파일명]에 [시스템] 공식을 설계해줘. [등급 조합]으로 예시 계산 포함.` |
| 정합성 검토 | `[폴더A]와 [폴더B]를 비교해서 불일치 부분을 목록으로 정리해줘.` |

---

## 주요 조작

| 기능 | 방법 |
|---|---|
| 파일 참조 | 입력창에 `@` 입력 후 파일 선택 |
| 수정 전 계획 확인 | 입력창 왼쪽 **Plan** 토글 활성화 |
| 비용 확인 | `/cost` 입력 |
| 대화 초기화 | `/clear` 입력 |

---

## 주의사항

- 수정 전 항상 diff 화면 확인 (Accept/Reject)
- CLAUDE.md 없으면 맥락 없는 일반 채팅과 동일
- AI가 만든 수치 공식은 **반드시 사람이 검증**
- 대화가 길어지면 앞 내용을 잊을 수 있으므로 중간에 파일로 저장
- 실시간 인터넷 정보 조회 불가 (KBO/MLB 실데이터는 별도 조사 필요)
